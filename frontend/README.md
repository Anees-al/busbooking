## Bus booking website

A modern bus booking web application built using
Mernstack The platform allows users to search routes,
view available buses, select seats, and book tickets easily.

## Features

User Registration(with password matching and unique email/phone check)
User login with JWT authentication
Secure password storing with bcrypt
Mobile responsive using tailwind CSS 
Get Buses by Routes and Operators


## Tech Stack

Frontend: React, Tailwind CSS. ,Framer Motion
Backend: Node.js, Express.js.
Database: MongoDB (via Mongoose).
Security: JWT (JSON Web Tokens), Bcrypt.
